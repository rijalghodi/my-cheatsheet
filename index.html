CSS Selector Reference

CSS Selectors

Selector Example Description
.class .contaier Selects all elements with class="container"
.class1.class2 .container.center Selects all elements with both container and center set within its class attribute
.class1 .class2 .container .card Selects all elements with card that is a descendant of an element with container
#id #firstname Selects the element with id="firstname"
* * Selects all elements
element p Selects all <p> elements
element>element div > p Selects all paragraph elements where the parent is a <div>> element
element+element div + p Selects the first <p> element that is placed immediately after <div>> elements
element1~element2 p ~ ul Selects every <ul> element that is preceded by a <p> element
